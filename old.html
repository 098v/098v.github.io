Branding
Bubble Fizz Concept Logo

Branding

Bubble Fizz Concept Logo

The Bubble Fizz logo that I had created was a concept logo for a made up Soda Pop company, the logo was made to show that with each bottle of Bubble Fizz you'll enjoy each and every sip, to accomplish this I had chosen bright and friendly colors that both adults and kids would find pleasing to the eye. With this design it also seperates itself from the other competitors in terms of colors and how the logo looks, many companies use colors that give off an exciting look using colors like green and green, but with the design of this logo it'll give off a nice and pleasent look something that they could pick up and drink during a hot day or just as a drink for their meal.

Before I had made this logo I had made two rough sketches of what the logo could look like which I felt like was enough because by the time I finished the second sketch I knew that I would use parts from both sketches, the bubbly background from the first sketch and the slanted text for "fizz". Once I had finsihed that I had started building the logo, the bubbles was just one bubble I had made and then copy and pasted multiple times but rotated to make each bubble different from the last. For the word Bubble I had used the Bubblegum font as I felt that it showed exactly what I had imagined the logo, but the font for Fizz I had made by hand because I hadn't found a font close enough to what I had drawn. Once everything was put together I added the colors and the underline under Fizz and this was the final product.

Cloud Login Screen Concept

This login and sign up form was created as a concept for a cloud based web/file hosting site that I had made up. The intent was to make a concept login/sign up portal for people where they could sign up using Facebook, Twitter or using their email. The reason for it not being landscape was so that the image could immitate the view from a mobile device like an iPhone or Android phone.

This entire project was created using Photoshop. I was going to create a quick sketch of the design but instead I had went straight to my laptop and started to work on it. For the colors I had decided to use monochromatic colors for the background, logo and login button, using whites and greys. I had chosen these colors so that the other two buttons would pop off the screen when looking at it

Bedroom
Bedroom

Cinema 4D

Bedroom made through Cinema 4D with lighting edited with Photoshop

This bedroom that I have made through Cinema 4D with some editing done in Photoshop was made to test my skills with 3D work, to see whether I can handle a large workload without loosing inspiration and the drive to complete this project. This project was sort of an on and off project because of school work and studying for exams taking the place of the time I would use specifically for this project, so it took me about 3 months but once I had all my school work done and my exams done I had finished this quickly. The lighting and textures were created using the help of tutorials provided by GreyScaleGorilla everything else was made using future found off of Google/Ikea.

Throughout this project each object had it's own file, the sizes were'nt made to scale as it would interfere with the setting as each object's size was very different from one another, the dresser on the side for example would have been longer than the bed and cut out of the render almost half way through it so I had made is a bit more shorter and the plant was also shrunk as the original size was quite large, almost the size of an adult. Once I had everything set to the sizes that I wanted for each object I had put everything together and hit render. Render time: 2-3 hrs.

3D Modelling
Abstract Owl

3D Modelling

Abstract Owl

This Owl was created when I wasnted to learn how to make 3D objects appear flat on a 3D surface, so I had went online found something similar to what I wanted to learn. once I had everything done I had changed the lighting and color through Photoshop and saved two copies, one for web and one regular png.

For this project I had first traced out parts of the owls body that had the most detail after each part was traced out I had imported them into Cinema 4D and molded them by using a sphere and added a "Random effector". Once I had one part done I had renderd out each part once and saved it as a PNG file, with everything done I had imported everything into one document and had scaled each part to fit the original image of the owl, with that done I had changed the hue and the color of the owl and created the eye. This entire projcet took about 2 days, render time: approx. 8 parts for about 1hr each, Photoshop editing: about 1-2hrs.

Photoshop
Exploding Sphere

Photoshop

Exploding Sphere

This exploding sphere that I had created was done for my George Brown portfolio which I had an interview for to see if I would get into the program. The project was created on the same day that I had started to make it.

For this project I had use both Cinema 4D and Photoshop to create this design. Through Cinema 4D I had made a sphere and increased it's segments so that when the object would later "explode" there would be a lot of peices flying out. After that I had added a Random Effector, Explosion FX, and added noise to the texture, after fooling around with the settings I had gotten a shape that I had really liked and rendered it out. Once the rendering was done I had brought the sphere into Photoshop, added some "grungey" brushes, some birds and added another sphere behind the first one but with a blurred effect on it. Finally I added a radial background and saved it as is.
